Group Buying Add-on - Loyalty IDs
=================================================

Option on Account page to add a loyalty number for a customer.

Installation
------------

1. Upload and activate plugin
2. Go to Group Buying > Addons and check the box for the Loyalty IDs module you want to activate.
3. Set an loyalty id on each customer account management page.
